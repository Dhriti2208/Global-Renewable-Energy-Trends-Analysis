# 🌍 Global Renewable Energy Trends Analysis (2010–2023)

## 📌 Overview
This project analyzes global renewable energy trends using real-world data from 2010 to 2023. It focuses on understanding how renewable energy capacity has evolved across countries and identifying key patterns in energy adoption.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly

## 📊 Key Features
- Data preprocessing and handling missing values
- Country-wise renewable capacity analysis
- Visualization using line charts, scatter plots, and maps
- Comparison of renewable energy sources (solar, wind, hydro)
- Analysis of relationship with GDP and population

## 🔍 Key Insights
- Rapid growth in renewable energy after 2015
- Solar and wind becoming dominant energy sources
- No strict correlation between GDP and renewable capacity
- High per-capita generation in smaller countries

## 📁 Project Structure
- `notebook.ipynb` → Main analysis
- `report.pdf` → Detailed explanation
- `data/` → Dataset (if included)
